Considero que la mejor forma de aprender es ensuciándose las manos con el hardware. Mi Homelab es el núcleo de mi entorno de desarrollo y pruebas, permitiéndome experimentar con arquitecturas de **Fog Computing** y orquestación de servicios.

### Hardware Spec:
* **CPU:** AMD Ryzen 5 9600X (6C/12T @ 3.9GHz)
* **Placa Base:** GIGABYTE B650 Eagle AX AM5
* **RAM:** 64GB DDR5 (Kingston Fury Beast + Corsair)
* **Storage:** 1TB NVMe Gen4 (Kingston NV3) + 4TB HDD (Seagate Barracuda)
* **Case:** Rack 4U Servidor
